  ```sh
  npm install -g truffle@5.1.4
  ```

### Installation

<!-- USAGE EXAMPLES -->
## Usage

Deploy the contracts on the local network provided by Gnache. 
```sh
   truffle migrate
   ```

Run the client app. 
```sh
   npm run start
   ```
